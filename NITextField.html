NITextField Class Reference

Overview

UITextField leaves a little to be desired on the visual customization front.

NITextField attempts to solve the most basic of those gaps so that the CSS subsystem can function properly.

applyNITextFieldStyleWithRuleSet:inDOM:

Applies the given rule set to this text field.

- (void)applyNITextFieldStyleWithRuleSet:(NICSSRuleset *)ruleSet inDOM:(NIDOM *)dom;
Discussion

This method is exposed primarily for subclasses to use when implementing the applyStyleWithRuleSet: method from NIStyleable.

applyNITextFieldStyleBeforeViewWithRuleSet:inDOM:

Applies the given rule set to this label.

- (void)applyNITextFieldStyleBeforeViewWithRuleSet:(NICSSRuleset *)ruleSet inDOM:(NIDOM *)dom;
Discussion

This method is exposed primarily for subclasses to use when implementing the applyStyleWithRuleSet: method from NIStyleable. Since some of the view styles (e.g. positioning) may rely on some label elements (like text), this is called before the view styling is done.

applyStyleWithRuleSet:forPseudoClass:inDOM:

Applies the given rule set to this text field but for a pseudo class.

- (void)applyStyleWithRuleSet:(NICSSRuleset *)ruleSet forPseudoClass:(NSString *)pseudo inDOM:(NIDOM *)dom;
Discussion

Thus it only supports the subset of properties that can be set on states of the button. (There's no fancy stuff that applies the styles manually on state transitions.

Since UIView doesn't have psuedo's, we don't need the specific version for UIButton like we do with applyButtonStyleWithRuleSet.
