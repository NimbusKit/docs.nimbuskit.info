Nimbus Launcher

Overview

Nimbus' Launcher view and related components.

A launcher view is best exemplified in Apple's home screen interface. It consists of a set of pages that each contain a set of buttons that the user may tap to access a consistent, focused aspect of the application or operating system. The user may swipe the screen to the left or right or tap the pager control at the bottom of the screen to change pages. A launcher also allows its buttons to be repositioned using a tap and hold gesture (though this has not been implemented yet in Nimbus' launcher, see the todo list for more details).

Example of an NILauncherViewController as seen in the BasicLauncher demo application.
