Version 0.5 API Changes
Version History

Version 0.5.0 of Nimbus was released on July 29, 2011.

This major version introduced the new Nimbus Overview, a debugging tool that shows detailed information about the state of your device and application in the device's status bar area.

The Overview added to the network photo album app.
