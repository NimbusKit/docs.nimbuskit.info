NIUserInterfaceString Class Reference

Overview

A very thin derivative of NSString that will track what user interface elements it has been assigned to and watch for updates via Chameleon.

attach:

Attach a string to a user interface element.

- (void)attach:(UIView *)view;
Discussion

Any changes to the string will be sent to the view until the view is dealloced or the string is detached. This method uses setText if available, else setTitle or asserts.

detach:

Detach a string from a user interface element.

- (void)detach:(UIView *)view;
Discussion

This does not "unset" the string value itself but it essentially stops tracking changes and frees internal structures.
